Assignment 1: Implementing PONG in Javascript with Observables.
Documentation can be found in the source code. Feel free to change some configurations in pong.html. Some configuration options are explained below.

  • best_of: How many points are required to win
  • tick_rate: How many times an Observable calls (in ms)
  • ai_difficulty: How closely the CPU paddle tracks the ball (default is 2.5)
You can also adjust player_score and cpu_score to give a headstart to the chosen paddle.
